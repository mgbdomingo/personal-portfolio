# Miguel Gabriel B. Domingo - Portfolio

Single-page portfolio website for Miguel Gabriel B. Domingo, GoHighLevel Expert and Automation Specialist.

## Stack
- Pure HTML5, CSS3, Vanilla JavaScript
- No frameworks, no dependencies, no build tools
- Google Fonts: Montserrat + Inter

## File Structure

## Deploy to GitHub Pages
```bash
git init
git add .
git commit -m "Initial portfolio site"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/miguel-portfolio.git
git push -u origin main
```
Then: Settings > Pages > Deploy from `main` branch.
